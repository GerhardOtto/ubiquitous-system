# Alt Coin Trader

A personal tool for tracking and managing alt coin trading activity.

## Stack

- **Next.js** — framework of choice; most comfortable working with it day-to-day
- **shadcn/ui** — pre-built, accessible components that speed up development without getting in the way
- **tweakcn** — theme editor on top of shadcn that makes visual customization quick and effortless
- **Vitest** — popular testing framework; still learning it, but it does the job

## Design Philosophy

Components are built with reusability in mind. UI pieces are kept small and focused so they can be composed across different parts of the app without duplication.

## Getting Started

```bash
npm install
npm run dev
```

Run tests:

```bash
npm test
```
